Sistema de Equação Linear - Álgebra Linear

Esta é a fórmula de uma equação de 1º grau, a qual também é chamada de equação linear, por além de possuir expoente igual a 1, também projeta uma Reta na sua exibição.

        A1X1 + A2X2 = B

Na introdução da álgebra linear, a fórmula geral a ser usada, é a seguinte:

        |A1X1 + A2X2 +....+AnXn = B1
        |C1X1 + C2X2 +....+CnXn = B2
        |.
     ---|.
        |.
        |K1X1 + K2X2 +....+KnXn = B3

Onde cada equação ipa projetar uma exata reta constante. Diante disto, existem 2 possbilidades para tais equações, onde O número de equações é igual ao número de variáveis ou O número de equações é menor que o número de variáveis (Caso o número de equações eja maior, a sua resolução irá gerar uma das duas opções citadas, portanto esta não é considerada). Além disso, as retas projetadas peas equações lineares, podem projetar os 3 possíveis campos paras as retas, onde as retas podem se tornar Paralelas (impossível e indeterminado), Cruzadas(possível e determinado) ou Sobrepostas(Possível e Indeterminado), dependendo do resultado gerado da equação, como os exemplos seguintes:

  1. Exemplo (Retas Cruzadas {Possível e Determinado}):
    2.  
                        | X1 - 2X2 = -1
                    |-X1 + 3X2 =  3

    Logo, é possível eliminar os X1 por serem contraditórios, gerando:

        
                    | -2X2 = -1
                    |  3X2 =  3

    Em uma equação linear, é necessário somar as equações para obter um valor para a variável X. Logo, realizando a soma, é gerado o seguinte resultado:

       
                    |X2 = 2

    A partir de então, basta escolher uma das equações, substituir o valor obtido da variável, pelas variáveis da equação e obter um resultado para a outra variável:

        
                    |X1 - 2.2 = -1

                     X1 - 4 = -1

                     X1 = -1 + 4

                     X1 = 3

    Portanto, os resultados das variáveis obtidas são X1 = 3 e X2 = 2. Pela equação ter gerado um valor para cada equação (ou seja, uma reta para cada equação), é dito que as retas das equações se CRUZAM.

  2. Exemplo (Retas Paralelas {Impossível e Indeterminado}):
    3.  
                        | X1 - 2X2 = -1
                    |-X1 + 2X2 =  3

    Logo, é possível eliminar os X1 e 2X2, por serem contraditórios, gerando:

                        |-1
                    | 3

    Por fim, realizando a soma dos dados, é obtido o valor 2. Portanto, as equações geraram apenas 1 valor para qualquer uma das variáveis. Dito isso, é etendido que as retas das equações são PARALELAS.

  3. Exemplo (Retas Sobrepostas {possível e Indeterminado}):
    4.  
                        | X1 - 2X2 = -1
                    |-X1 + 2X2 =  1

    É claro de se ver que todos os valores se contradizem, podendo então tornar a equação inteira igual a zero e quando as Retas das equações geram um mesmo valor nulo para as variáveis, é dito que as retas das equações estão SOBREPOSTAS.

Porém, perceba que todas as equações citadas acima são sobre apenas 2 retas e obviamente não existem apenas 2 retas para as equações lineares, 3,30,300 ou infiitas retas na equação. Mas realizar os cálculos para mais de 2 retas e torna um pouco mais complexo e é ai onde entra a Álgebra linear.

Em caso de aparecerem 3 ou mais equações lineare em toda a equação, é sugerido transforma-la em uma matriz, sendo o primeiro passo Gerar uma matriz com os valores dados, segundo passo Gerar uma matriz com as igualdades das equações e terceiro passo Concatenar as matrizes obtidas. O resultado da concatenação, é chamdo então de Matriz Aumentada , como o exemplo seguinte:

        
            |A1X1 + A2X2 +....+AnXn = D1
            |B1X1 + B2X2 +....+BnXn = D2
            |.
         ---|.
            |.
            |C1X1 + C2X2 +....+CnXn = Dn
         
        Passo I 
        Gerar a matriz com os valores:
            
            A1  B1  ....   C1
            A2  B2  ....   C2
            An  Bn  ....   Cn
            
        Passo II 
        2ª matriz com os valores da igualdade:
        
            D1
            D2
            Dn
            
        Passo III    
        Concatenação das matrizes (Matriz Aumentada):
        
            A1  B1  C1  D1
            A2  B2  C2  D2
            An  Bn  Cn  Dn

Portanto, toda equação linear pode gerar uma matriz aumentada e vice versa, a qual é recorrida quando há mais de 2 equações em toda a equação.
