Potência de uma matriz - Álgebra Linear

Só é possíel múltiplicar uma matriz se Anxm . Anxm onde M = N (ou seja, matriz quadrada), ou seja, dado uma matriz A, a potência A^x só é possível se A for uma matriz quadrada. Como no exemplo a seguir: 

          A^2 = [ 1  4/3  6/5 ]   [ 1  4/3  6/5 ]
                [ 2  5/2  1/4 ] X [ 2  5/2  1/4 ]
                [ 3   7  Raiz3]   [ 3   7  Raiz3]

Logo, o primeiro elemento é a soma da multiplicaçãp da 1ª linha com a 1ª coluna, onde com o seguimento do exemplo aneterior:

 
            A^2 = [(1.1 + 4/3.2 + 6/5.3)    (1.4/3 + 4/3.5/2 + 6/5.7)]

Sendo este cálculo, o valor dos elementos 1 e 4 da matriz A^2. Então, basta seguir o mesmo método para os demais elementos, para cada elemento. Portanto, a representação final da matriz anterior, é a seguinte:

            A^2 = [(1.1 + 4/3.2 + 6/5.3)    (1.4/3 + 4/3.5/2 + 6/5.7)   (1.6/5 + 4/3.1/4 + 6/5.Raiz3)]
                  [(2.2 + 5/2.2 + 1/4.3)    (2.4/3 + 5/2.5/2 + 1/4.7)   (2.6/5 + 5/2.1/4 + 1/4.Raiz3)]
                  [(3.2 + 7.2 + Raiz3.3)    (3.4/3 + 7.5/2 + Raiz3.7)   (3.6/5 + 7.1/4 + Raiz3.Raiz3)]
                  
        Onde    
            
            A^2 = [(1 + 8/3 + 18/5 )    (4/3 + 10/3 + 42/5)   (6/5 + 1/3 + 6Raiz3/5)]
                  [(2 +  5  +  3/4 )    (8/3 + 25/4 +  7/4)   (12/5 + 5/6 + Raiz3/4)]
                  [(3 + 14 + 3Raiz3)    (4 + 35/2 + 7Raiz3)   (18/5 + 7/4 +    3   )]
                  
        Logo
        
            E1 = 1 + 8/3 + 18/5 = 109/15
            E2 = 4/3 + 10/3 + 42/5 = 196/15
            E3 = 6/5 + 1/3 + 6Raiz3/5 = 23+18Raiz3/15
            E4 = 2 +  5  +  3/4 = 31/4
            E5 = 8/3 + 25/4 +  7/4 = 125/12
            E6 = 12/5 + 5/6 + Raiz3/4 = 125+10Raiz3/40
            E7 = 3 + 14 + 3Raiz3 = 17+3Raiz3
            E8 = 4 + 35/2 + 7Raiz3 = 43+14Raiz3/2
            E9 = 18/5 + 7/4 + 3  = 167/20
            
        Por fim, seu resultado é o seguinte:
        
            A^2 = [109/15     190/15      23+18Raiz3/15 ]
                  [31/4       125/12      125+10Raiz3/40]
                  [17+3Raiz3  43+14Raiz3  167/20        ]

Transposta de Matriz

Basicamente, a transposição de uma matriz, é alterar a posição de seus valores, onde o que é linha se torna coluna e vice-versa. Como no exemplo:

            A = [ X  1 ] -----> A = [X  Y  Z]
                [ y  2 ]            [1  2  3]
                [ Z  3 ]

Perceba que algo importante a ser destacado, é que a quantidade de linhas se torna a mesma quantidade de colunas e vice-versa. Além disso, esse estudo, possui 4 teoremas que devem/podem ser respeitados, os quais podem facilitar a resolução de algum problema. Sendo eles:

  1. (A^t)^t = A
  2. (A+B)^t = A^t + B^t
  3. B.(A^t) = (B.A)^t
  4. (A.B)^t = A^t . B^t

Onde t é a transposta, como no teorema 2, onde a transposta de (A+B) é igual a transposta de A + a transposta de B.

Matriz Inversa

Uma identidade de uma matriz é representada de forma que os valores de sua multiplicação, se tornem os mesmos valores originais, ou seja, os valores devem ser 1 e 0, como a seguir: 

            A = [ 1  3 ] 
                [ 2  4 ]
                
        Logo, sua identidade seria:
        
            A = [ 1  0 ]
                [ 0  1 ]

Dado a matriz A, sua inversa é A^-1, tal que A.A^-1 é uma identidade. Assim como A deve ser quadrada, onde Anxm.A^-1nxm=In.

Para se cálcular a inversa de uma matriz, alguns passos devem serseguidos, sendo eles:

  1. É preciso atribuir, convencionalmente, icóguinitas para a matriz inversa.
  2. É preciso efetuar o produto A.A^-1 e igualar a I(identidade).
  3. É preciso montar um sistema de equações e resolve-lo.
  4. É preciso escrever a inversa com os valores das icóguinitas obtidas.

Como no exemplo a seguir:

            A = [ 1  3 ]
                [ 2  4 ]
                
        Passo 1 - Atribuir icóguinitas para a matriz inversa
        
            A = [ 1  3 ] x [ a  b ]
                [ 2  4 ]   [ c  d ]
                
        Passo 2 - Efetuar o produto de A.A^-1 e igualar a identidade
        
            A = [ 1  3 ] x [ a  b ] = [ 1  0 ]
                [ 2  4 ]   [ c  d ]   [ 0  1 ]
                
            Logo
            
            A = [ a+3c    b+3d ] = [ 1  0 ]
                [ 2a+4c  2b+4d ]   [ 0  1 ]
                
        Passo 3 - Montar um sistema de equações
        
            A = [ a+3c    b+3d ] = [ 1  0 ]
                [ 2a+4c  2b+4d ]   [ 0  1 ]
                
            onde
            
            a+3c  = 1
            b+3d  = 0
            2a+4c = 0
            2b+4d = 1
            
        Perceba que uma equação linear foi formada, podendo então, realizar uma Matriz aumentada para resolver
        Onde essa matriz aumentada deve ter um corpo 4x5 ( 4 Linhas - Qtde de equações e 5 colunas - Qtde de colunas no Passo 2)
        Logo:
        
            [ 1 0 3 0 1]
            [ 0 1 0 3 0] ----> Matriz Aumentada
            [ 2 0 4 0 0]
            [ 0 2 0 4 1]
